# AnimePlayer

## A Sci-Fi Cyberpunk Video Player for Android

AnimePlayer is a futuristic video player for Android, designed with a "Sci-Fi Cyber" aesthetic. It's built with Jetpack Compose and features a unique, immersive UI that's perfect for watching your favorite videos.

### Features

*   **Sci-Fi Cyber UI:** A custom-designed UI with a dark, futuristic aesthetic, complete with glowing neon accents and a holographic feel.
*   **"Liquid Pulse" Loading Animation:** A stunning loading animation that sets the tone for the app's futuristic design.
*   **"Bento Grid" Library:** A dynamic, multi-column grid layout for browsing your video library.
*   **"Glassmorphism 2.0" Player Controls:** Floating, blurred control panels with glowing neon borders that give the player a holographic look.
*   **Dynamic Gradient Seek Bar:** A futuristic seek bar that animates with a flowing gradient.
*   **Remember Playback Position:** The app remembers where you left off in a video, so you can easily resume watching.
*   **Picture-in-Picture (PiP) Mode:** Watch your videos in a small, floating window while you use other apps.
*   **Advanced Gesture Controls:** Double-tap to seek and swipe to control brightness and volume.
*   **"Open with" Functionality:** Open videos from other apps directly in AnimePlayer.
